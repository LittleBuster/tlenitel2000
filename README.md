Tlenitel 2000
==================================

This application created for quizes on radio anison.fm<br>
Tlenitel 2000 difficult to find images in google search.

Dependencies:
````
MinGW32, MSSYS, OpenCV library, Python 3.4, PyQt4
````

To make libtlen you need write:
````
cd libtlen
make
````
